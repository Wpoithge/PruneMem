# Overview

PruneMem is a structured memory plugin that combines:

- layered memory organization
- registry-driven governance
- full lifecycle correction and maintenance
- pluggable retrieval/model interfaces

It is designed as a public, portable implementation rather than a mirror of any private workspace.

## Public demo flow

The repository now includes a public sample flow that can run in mock mode:

1. `session-packet.json`
2. `run-extract`
3. `run-judge`
4. `update-registries`
5. `maintain`

This allows users to understand the pipeline end-to-end before connecting a real provider.
